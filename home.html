What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage and also through JavaScript, we can interact with the DOM to dynamically update the page: like we can add, remove, modify elements and change styles or attributes. Using the DOM a webpage can update content dynamically.

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is a technique in web development where you attach a single event listener to a parent element.When an event is triggered on an element, it first runs the event handlers on that element, then bubbles up the DOM tree, triggering event handlers on its ancestors.Instead of attaching event listeners to multiple child elements, you add a single event listener to a parent element. When an event occurs on a child element, it bubbles up to the parent, where the event listener can capture it.
